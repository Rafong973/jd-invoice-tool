# 京东发票管理工具

一款轻量级桌面客户端，帮助您高效管理京东发票、批量换开发票。

## ✨ 功能亮点

- 📋 **发票列表**：一键查看已开具的所有发票，支持滚动加载更多
- 🔄 **换开发票**：按开票方自动分组，自由勾选订单组合，快速发起换开
- 🏢 **抬头管理**：预设个人/公司发票抬头，支持设置默认抬头
- 💾 **本地存储**：所有数据保存在本地，安全可控

## 🚀 快速上手

### 1. 首次登录

1. 打开软件，点击 **「打开京东登录」** 按钮
2. 系统会自动打开浏览器，请在浏览器中完成京东账号登录
3. 登录成功后，按 `F12` 打开开发者工具 → 切换到 **Network（网络）** 标签
4. 刷新页面，在左侧请求列表中点击任意一个京东域名的请求
5. 在右侧 **Request Headers（请求标头）** 中找到 `Cookie` 字段，完整复制其值
6. 回到客户端，将 Cookie 粘贴到文本框中，点击 **「保存并登录」**

> 💡 **提示**：Cookie 会保存在本地，下次打开软件无需重新登录。如果登录失效，重新粘贴即可。

### 2. 查看发票列表

登录成功后，自动进入发票列表页：

- 页面会自动加载第一页发票数据
- **向下滚动到底部** 会自动加载下一页
- 每张发票显示：发票抬头、状态、订单号、商品名称、金额
- 点击 **「下载」** 按钮可将发票 PDF 保存到本地

### 3. 换开发票

1. 点击顶部导航栏的 **「换开发票」**
2. 点击 **「获取可换开发票」**，系统会自动拉取所有可换开的订单
3. 订单会按 **开票方** 自动分组显示
4. 在每个分组内：
   - 勾选需要换开的订单（可全选或部分勾选）
   - 点击 **「申请换开」** 按钮
5. 在弹出的确认窗口中：
   - 选择发票抬头（默认使用已设置的默认抬头）
   - 确认订单信息和金额无误后，点击 **「确认换开」**

> ⚠️ **注意**：换开只能在同一开票方内进行，不可跨开票方合并。

### 4. 管理发票抬头

在 **「设置」** 页面可以管理发票抬头：

1. 切换到 **「个人」** 或 **「公司」** 标签
2. 填写抬头名称和接收邮箱（必填），公司类型还需填写税号
3. 勾选 **「设为默认抬头」** 可将其设为换开时的默认选项
4. 点击 **「添加」** 保存

已添加的抬头支持：
- 设为默认
- 删除

## ❓ 常见问题

### Q: Cookie 多久会过期？
A: 京东 Cookie 通常有效期为 30 天左右。过期后重新登录并粘贴新 Cookie 即可。

### Q: 如何获取 Cookie？
A: 登录京东后，按 F12 打开开发者工具 → Network → 刷新页面 → 点击任意请求 → 在 Request Headers 中复制 Cookie 值。

### Q: 换开发票后多久能收到新发票？
A: 换开申请提交后，通常 1-3 个工作日内会处理完成，新发票会发送到您设置的邮箱。

### Q: 数据保存在哪里？
A: 所有数据（Cookie、发票抬头）均保存在本地，不会上传到任何服务器。
